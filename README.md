# DevLinks API

A production-grade bookmark manager REST API. Save URLs, add tags and notes, search by keyword or tag.

Built as a capstone project to consolidate everything learned in Phase 1 — Git, FastAPI, Docker, testing, and CI/CD — into a single cohesive, real-world codebase.

## Why this project

I learned Git , Docker , containerisation , FastAPI , SQL and Github Actions. This project will be a test for me how much i have learned so far. Stay Tuned.

## Stack

- **Python 3.11+** with FastAPI and Pydantic v2
- **SQLAlchemy** ORM with SQLite (dev) / PostgreSQL (prod)
- **JWT** authentication
- **pytest** test suite targeting ≥80% coverage
- **Docker** for containerisation
- **GitHub Actions** for CI/CD
